# Portfolio (Next.js 14, App Router)

Minimal, production-ready portfolio built with **Next.js 14**, **TypeScript**, and **Tailwind CSS**.

## Tech stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS

## Project structure

- `app/` – App Router entry, layout and pages
- `components/ui/` – Reusable presentational components (buttons, cards, layout primitives)
- `components/sections/` – Page sections (hero, about, projects, experience, contact)
- `lib/` – Utilities and shared helpers
- `data/` – Structured content for projects, experience, and links
- `types/` – TypeScript types used across the app

## Getting started

```bash
npm install
npm run dev
```

Then visit `http://localhost:3000`.
